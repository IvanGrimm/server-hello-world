## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/IvanGrimm/simple-web-server.git
    ```

2. Navigate to the project directory:
    ```bash
    cd simple-web-server
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

To start the server, run:
```bash
npm start
